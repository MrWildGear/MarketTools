# Icon Generation

This directory contains scripts to generate placeholder icons for Elinor Reloaded.

## Using Node.js with Sharp (Recommended)

```bash
pnpm install
pnpm generate-icons
```

## Alternative: Using node-canvas

If sharp doesn't work, you can use the simpler node-canvas version:

```bash
pnpm generate-icons-simple
```

Note: This requires the `canvas` package. Install with `pnpm add -D canvas`.

## Manual Creation

If the scripts don't work, you can manually create icons:

1. Create a simple icon image (256x256px) with:
   - Blue background (#3b82f6)
   - White "E" letter in the center
   - Rounded corners

2. Save/export to these locations:
   - `src-tauri/icons/32x32.png`
   - `src-tauri/icons/128x128.png`
   - `src-tauri/icons/128x128@2x.png` (256x256)
   - `src-tauri/icons/icon.png` (256x256)
   - `src-tauri/icons/icon.ico` (Windows icon)
   - `src-tauri/icons/icon.icns` (macOS icon)

## Note

The `.ico` and `.icns` files generated by the scripts are placeholders (PNG format).
For production builds, you may want to convert them to proper ICO/ICNS format using:
- Online converters
- ImageMagick
- icnsutil (for macOS)
- ImageIco (for Windows)
